Welcome to the comprehensive guide collection for **VEIN**, a survival sandbox set in the northern reaches of New York State. This repository contains both gameplay primers for survivors and technical documentation for server administrators.

This is a collection of helpful tips provided by the community.

***Disclaimer:***  As of this writing (2026-05-08) the game is currently running on version 0.23 (*stable*), with 0.24 being on the *experimental branch.*
